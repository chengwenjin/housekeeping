# 家政小程序 - 项目设计总览

## 📋 项目简介

家政小程序是一个家政服务撮合平台，连接有家政服务需求的用户和提供服务的服务者。基于微信小程序生态，提供便捷的需求发布、在线接单、订单管理等服务。

**项目名称**: 家政速帮  
**版本**: v1.0  
**设计日期**: 2026-03-26

## 🎯 核心功能

1. **用户登录**: 微信扫码一键授权登录
2. **需求浏览**: 首页展示所有家政需求，支持筛选和搜索
3. **需求详情**: 查看需求详细信息，包括服务时间、地点、价格等
4. **在线接单**: 服务者可快速接单
5. **需求发布**: 用户可发布自己的家政需求
6. **订单管理**: 
   - 查看我发布的需求及接单情况
   - 查看我接的订单
7. **社交功能**: 
   - 浏览足迹记录
   - 关注其他用户
8. **评价系统**: 双向评价机制

## 🔐 安全设计

### 认证机制
- **JWT Token**: 有效期 7 天，支持刷新
- **微信授权**: code2session 获取 openid
- **权限控制**: RBAC 基于角色的访问控制

### 数据安全
- SQL 注入防护 (ORM 参数化)
- XSS 防护 (输入过滤)
- CSRF 防护 (Token 验证)
- 敏感信息加密存储

### 业务安全
- 需求审核机制 (可选)
- 用户举报功能
- 黑名单系统
- 风控规则引擎

## 📈 性能优化

### 缓存策略
- 热点数据 Redis 缓存 (30 分钟 TTL)
- 首页数据预加载
- 图片 CDN 加速

### 数据库优化
- 索引优化 (常用查询字段)
- 读写分离 (主从复制)
- 分页查询 (避免全表扫描)

### 接口优化
- 按需查询 (避免 SELECT *)
- 批量操作 (减少往返次数)
- 异步处理 (消息队列)
