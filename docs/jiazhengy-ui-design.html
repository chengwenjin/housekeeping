📐 设计规格说明

🎨 视觉风格

  • 主色 #FF6B35 温暖橙，传递活力与信任
  • 辅色 #2EC4B6 清爽青绿，差异化强调
  • 渐变 橙→琥珀，活泼不失稳重
  • 圆角体系 8/12/16/20px，友好柔和
  • 卡片阴影 低对比度，层级清晰
  • 字体系统 PingFang SC，清晰可读

📱 交互设计

  • 底部Tab 5项，中间凸起发布按钮
  • 首页搜索置顶，快捷分类横滑
  • 需求卡片信息层级明确：标题→描述→元信息→价格/接单
  • 详情页黏性底栏，随时接单
  • 发布流程分区块，逐步引导
  • 接单/已接单 状态色彩区分

♿ 可访问性

  • 主文本对比度 ≥ 7:1 (WCAG AAA)
  • 辅助文本对比度 ≥ 4.5:1 (AA)
  • 最小点击目标 44×44px
  • 状态信息用色彩+文字双重传达
  • 表单项含清晰 label 和 placeholder
  • 价格金额加单位说明避免歧义
