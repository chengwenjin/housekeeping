# 家政小程序管理后台前端

基于 Vue 3 + Element Plus 的家政服务小程序管理后台系统。

## 技术栈

- **框架**: Vue 3.4
- **UI 组件库**: Element Plus 2.5
- **状态管理**: Pinia 2.1
- **路由**: Vue Router 4.2
- **HTTP 请求**: Axios 1.6
- **构建工具**: Vite 5.0
- **图表**: ECharts 5.4
- **时间处理**: Day.js

## 功能模块

- ✅ 用户登录认证
- ✅ 首页数据概览
- ✅ 用户管理
- 🚧 需求管理 (开发中)
- 🚧 订单管理 (开发中)
- 🚧 评价管理 (开发中)
- 🚧 分类管理 (开发中)
- 🚧 数据统计 (开发中)
- 🚧 系统配置 (开发中)
- 🚧 操作日志 (开发中)

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

启动后访问：http://localhost:3000

### 生产构建

```bash
npm run build
```

### 预览构建结果

```bash
npm run preview
```

## 项目结构

```
管理后台前端/
├── public/              # 静态资源
├── src/
│   ├── assets/         # 项目资源
│   ├── components/     # 公共组件
│   ├── layouts/        # 布局组件
│   ├── router/         # 路由配置
│   ├── store/          # 状态管理
│   ├── utils/          # 工具函数
│   ├── views/          # 页面组件
│   ├── App.vue         # 根组件
│   └── main.js         # 入口文件
├── index.html
├── package.json
└── vite.config.js
```

## API 代理配置

开发环境下，所有 `/api` 开头的请求会代理到后端服务：

- 目标地址：http://localhost:8080
- 修改配置请编辑 `vite.config.js`

## 默认测试账号

根据后端 API 设计文档：

- 用户名：admin
- 密码：需要咨询后端开发人员
